# term-chart
Header only c++ library for drawing simple charts in the terminal. Should work from C++14 onwards

## Basic usage
```cpp
std::vector<int> values = { 1, 2, 3, 4, 3, 2, 1, 2, 3, 4, 5 };
std::cout << term_chart::render(4, values);
```
